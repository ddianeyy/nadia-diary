# Her Doll Diary

A small static website for a private doll collection and birthday page.

## How to open

Open `index.html` in a browser.

## How to add dolls now

Go to `collection.html`, choose a photo, fill in the details, and click
`Generate Caption`. The site stores dolls in your browser only. Use
`Export Backup` after adding real dolls so you can keep a copy.

## Free hosting recommendation

Use GitHub Pages or Vercel for free hosting. For your case, Vercel is the
easiest later if you want a nicer upload flow, but GitHub Pages is enough for
this static version.

## About automatic names and captions

This first version avoids paid AI. It guesses a doll name from the photo file
name and generates a cute caption from the fields you type. True name extraction
from text inside images needs OCR or AI later.
